Метод get:
- пустой ответ + проверка из бд
- единичный ответ + проверка из бд
- массивный ответ + проверка из бд
- вызов по id
- ответ  одинаковыми именем и почтой + проверка из бд
- ответ с пустыми полями + проверка из бд
- ответ с None + проверка из бд
- большое количество пользователей + проверка из бд

Метод post:
- Создание пользователя + проверка из бд
- Создание пользователя с полями None + проверка из бд
- Создание польователя с пустыми полями + проверка из бд
- Создание пользователя, который уже есть в БД
- Создание пользователя, только с одним полем + проверка из бд
- Ошибки?

Метод put:
- Изменение пользователя, который есть в БД + проверка из бд
- Изменения пользователя, которого нет в БД + проверка из бд
- Изменение двух полей + проверка из бд
- Изменение одного поля + проверка из бд
- Установка пустых значений + проверка из бд
- Установка пользователя + проверка из бд
- Пустое изменение без id + проверка из бд

Метод Delete:
- Удаление пользователя, который есть в БД + проверка из бд
- Удаление пользователя, которого нет в БД + проверка из бд
- Удаление без id
- Удаление с несколькими id
